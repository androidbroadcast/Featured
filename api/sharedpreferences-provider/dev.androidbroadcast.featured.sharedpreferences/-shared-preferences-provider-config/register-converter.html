registerConverter

fun <T : Any> registerConverter(klass: KClass<T>, converter: TypeConverter<T>)

Registers a TypeConverter for klass, enabling set and get for custom types (e.g. enums) that are serialized as strings in SharedPreferences.

Must be called before any set or get call for the corresponding type. Prefer the inline overload registerConverter to avoid passing KClass explicitly.

Parameters

T

The non-null type to register.

klass

The KClass of the type.

converter

The TypeConverter that serializes/deserializes T as a String.
