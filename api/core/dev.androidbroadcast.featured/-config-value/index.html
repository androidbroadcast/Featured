ConfigValue

data class ConfigValue<T : Any>(val value: T, val source: ConfigValue.Source)

A snapshot of a single configuration parameter's resolved value together with its origin.

ConfigValues produces ConfigValue instances when you call ConfigValues.getValue or collect from ConfigValues.observe. The source property lets callers distinguish between a hard-coded default, a locally persisted override, and a value fetched from a remote configuration service.
